2020年新⼯科联盟-Xilinx暑期学校（Summer School）项目介绍

项目概要
  摄像头识别纯底色的数字，并将数字显示（LCD、OLED、显示屏）
  
使用工具版本
  Vivado2018.3 、 Matlab 2020 、 Arduino

  外设：OV5647模组CSI接口树莓派摄像头模组、液晶显示屏
